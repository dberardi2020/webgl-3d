# webgl-3d

A 3D mesh viewer built with raw WebGL. College graphics course project.

**[Live Demo](https://dberardi2020.github.io/webgl-3d/)**

## Features
- Load `.ply` mesh files and render them in 3D
- Real-time rotation, translation, and scaling
- Lighting and surface normals
- Pulse animation effect

## Controls
| Key | Action |
|-----|--------|
| Arrow keys | Rotate mesh |
| `W/A/S/D` | Translate mesh |
| `P` | Toggle pulse animation |
